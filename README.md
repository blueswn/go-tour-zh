
# 欢迎使用 Go 指南（Go 语言之旅）

Go 语言的交互式简介

* 第一节覆盖了基本语法及数据结构，
* 第二节讨论了方法与接口，
* 第三节则简单介绍了 Go 的并发原语。

每节末尾都有几个练习，你可以对自己的所学进行实践。 你可以 在线学习 或 安装到本地。
